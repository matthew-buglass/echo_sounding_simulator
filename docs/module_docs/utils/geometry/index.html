Geometry

Declares and maintains utility function for geometric abstractions

find_x_y_theta(p1, p2, p3)

Finds the absolute angle (in radians) of intercept of the lines that pass through p1-p2 and p2-p3, intercepting at v2. Args: p1: an endpoint in the angle p2: The intercept point p3: an endpoint in the angle

Returns:

Type Description
float

The radians angle, theta, between p1-p2-p3

get_x_y_rotated_vector(vector, theta)

Rotates a vector by theta about the origin Args: vector: a numpy array representation of a vector [x y] theta: the angle, in radians to rotate by

Returns:

Name Type Description
rotated_vector ndarray

the vector rotated about the [x y] origin

line_sign(point, line_start, line_end)

Calculates whether a point lies above or below a line between two points

Parameters:

Name Type Description Default
point tuple[float, float]

The point of interest [x y]

 required
line_start tuple[float, float]

One side of the line [x y]

 required
line_end tuple[float, float]

The other side of the line [x y]

 required

Returns:

Name Type Description
sign float

1 if above the line, -1 if below, 0 if on the line

point_in_tri(point, v1, v2, v3)

Calculated whether an [x y] point is within the vertical slice of the triangular face created by the [x y z] vertices.

Parameters:

Name Type Description Default
point tuple[float, float]

point of interest

 required
v1 tuple[float, float, float]

[x y z] vector

 required
v2 tuple[float, float, float]

[x y z] vector

 required
v3 tuple[float, float, float]

[x y z] vector

 required

Returns:

Name Type Description
in_tri bool

a boolean of whether the point is withing the [x y] bound of the face

triangular_plane_intercept(x, y, v1, v2, v3)

Calculates the z value of a point with coordinates [x y] on the plane defined by v1, v2, and v3.

Parameters:

 required

[x y z] vertex

 required
v2 tuple[float, float, float]

[x y z] vertex

 required
v3 tuple[float, float, float]

[x y z] vertex

 required

Returns:

Name Type Description
z float

the z position of the projected points x and y
